# Calendar2026_Csharp_app
アプリ名【カレンダー2026】: UI/UX学年発表会で約140人中2位の評価を頂きました。

※わかりずらいため大変恐縮ですが、「Form3.cs→Form1.cs→Form2.cs」の流れでコードを読むことができます。

2025年6月に制作しました。

------------------------------------------------------------------------------------------------------------------------------

「概要」

本プロジェクトは日々の業務に疲れた人へ、癒しを届けることを目的に制作しました。

このミッションを達成するために、全て自作の画像を使い、情景にあったBGMを採用しています。

※BGMは「DOVA-SYNDROME YouTube Official」様で公開されているものを使用させていただいております。

------------------------------------------------------------------------------------------------------------------------------

「システムの内容」

本アプリの主な機能は2026年のカレンダーです。

2026年でカレンダーを制作した理由は、当時2025年の6月で、すでに半年経っていたため翌年のものを作ろうと考えたためです。

------------------------------------------------------------------------------------------------------------------------------

「ターゲット層」

疲れがたまり、癒しを求めている20～30代。

------------------------------------------------------------------------------------------------------------------------------

「作成背景」

もともとは学内の制作課題として「四季を表示する」というテーマで、UI/UXが評価される発表会でした。

そこで私は、ただ見やすく、操作しやすいだけではなく、さらに感動を届けられるようなものを作ろうと思いました。

デザインを学んでいた身なので、これは負けられないと思い、誠心誠意で制作にあたる期間でした。

「機能一覧」

・カレンダー機能：シンプルで丸みのある数字が特徴です。祝日はもちろん、その月の行事まで記載されています。

・祝日機能：ホーム画面で選択されている月の祝日や行事を詳細に教えてくれる機能です。私なりに解釈し記載しています。

・チュートリアル機能：インコが次への操作方法を教えてくれるので、迷わずに操作することができます。
また、チュートリアルはアプリ内で何度でも再表示することができます。

・音楽操作機能：簡易的ではありますが、ホーム曲を選択する機能と音楽のみミュートにする機能がございます。
(効果音もミュートにしたい場合は、パソコンのミュート機能を使うと解決するので実装していません。)

「工夫した点」

・操作はユーザーが直観的に触れるように設計しました。具体的には矢印ボタンで月を変更できるようにしていたり、
季節へカーソルを合わせると月が表示されて選択できるようにしていることなどがあります。

・見栄えも気にしながら制作していたため、ボタンやラベルなどは全て画像を使い制作しています。正直システムより画像作成に時間がかかっていました。

※余談になりますが、イラストは計5枚用意しており、1枚あたり1週間程度で制作しています。

・イラストで工夫した話になりますが、四季のイラストにはすべて列車が入っています。これは季節の移り変わりと切なさを示唆するために表現しました。

------------------------------------------------------------------------------------------------------------------------------

「課題・改善点」

・このアプリではカレンダーを画像で作成したため、2026年以外に対応していませんでした。
また、当時データベースと連携させることができたなかったため、データの保存は簡易的なことしかできなかったことなどが課題だと感じました。

・コードに無駄が多く、特に関数を使っていなかったり変数の名前がわかりずらいと感じました。今後の制作では気をつけたいと思います。

・当時、動画編集の技術がなく、動画時間が長くなってしまったことで印象が落ちたことをもったいなく思いました。実際に評価をしたのはIT学科の
学生ではなく、デザイン学科の生徒だったので、見せ方次第では学年で1位になっていてもおかしくない作品だったと思うので悔しく思います。

「導入手順」

1. 「Visual Studio 2022」をダウンロードし、「Windows フォームアプリケーション(.NET Framework)」を使用できる状態にします。

2.　GitHubから本アプリをダウンロードします。その後Zipファイルの状態で右クリックし、プロパティを選択します。

3. 　下の方にあるセキュリティで「許可する」チェックボックスをクリックし適用します。(なければそのまま「OK」を押して問題ありません)

4. 　Zipファイルを解凍し、「0H03018_重永敏英_uiux01.sln」をダブルクリックし起動します。
(もしアプリケーションの設定がが表示される場合は念のため「Visual Studio 2022」に設定をお願いします)

5. 　「▷ 開始」を押すとアプリが立ち上がります。

※もし、この手順で動作しなかった場合、申し訳ありませんが、デモ動画を参照していただけると幸いです。
